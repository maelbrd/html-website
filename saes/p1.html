Packet Tracer

Packet Tracer est un logiciel de simulation de matériel réseau Cisco
Il permet de concevoir des architectures réseau virtuel, afin que :

  • Les étudiants se forment au réseau sans matériel physique
  • Les professionnels peuvent tester certaines architectures avant de les mettre en place réellement.
  • Le tout se fait exclusivement avec des équipements virtuels Cisco.

    Projet routage

    Cette partie du projet nous a appris à faire fonctionner une infrastructure avec plusieurs réseaux.
    Tous les réseaux devaient communiquer entre eux grâce à la configuration des routeurs.
    Pour tout faire fonctionner, j'ai dû :

  • Configurer les routes pour que les réseaux puissent communiquer entre eux
  • Configurer les interfaces des routeurs avec les bonnes adresses Ip
  • Configurer l'interface de gestion des switchs

    • Après test tous les équipements ping entre eux.

    Commutation vlans

    Cette partie du projet nous a appris à faire du routage Inter-Vlans.


    Pour cela j'ai du :
  • Configurer les sous-interfaces des routeurs avec les bonnes adresses Ip
  • Configurer les interfaces des switchs avec les bonnes adresses Ip
  • Configurer les VLANs sur les switchs
  • Configurer le routage Inter-Vlans
  • Autoriser les bons Vlans dans le mode trunk

    • Une fois le routage réalisé tout les vlans peuvent communiquer.

    Compétences acquises

    J'ai pu développer plusieurs compétences comme :

  • Configuration des switchs et routeurs : Capacité à configurer les switchs et routeurs en utilisant des commandes spécifiques
  • Principe du Spanning-Tree : Compréhension des concepts du Spanning-Tree Protocol (STP) et de ses variantes pour éviter les boucles dans les réseaux.
  • Routage Inter-Vlans : Capacité à configurer le routage entre différents réseaux virtuels (VLANs) pour permettre la communication entre eux.
  • Configuration des trunks : Capacité à configurer les trunks, qui permettent le transport de plusieurs VLANs sur une seule connexion physique.
